Game of Cribbage

The following project is the simplified version of the game called Cribbage. The game is played with a standard 52-card deck. During the game, two players are both dealt five cards from the deck according to specific rules.

The technical part of this assignment was training to create nested lists to represent cards within the deck and each player's hand. By creating functions deal_card, score_hand and play I was able to write a program that determines the winner based on specific input.
